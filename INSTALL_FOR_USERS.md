# Jobhandy Extension - User Install Guide

This guide is for users who only need to install and run the extension.

## 1) Download

1. Open the GitHub repository page.
2. Click `Code` -> `Download ZIP`.
3. Save the ZIP file to your computer.
4. Right-click ZIP -> `Extract All...`.

## 2) Install in Chrome

1. Open Chrome and go to `chrome://extensions`.
2. Turn on `Developer mode` (top-right switch).
3. Click `Load unpacked`.
4. Select the extracted `jobhandy` folder (the folder that contains `manifest.json`).

The extension is now installed.

## 3) Run the tool

1. Open the company brand-exemptions page and log in normally.
2. Click the Jobhandy extension icon.
3. Upload your Excel file (`.xlsx`) or CSV/TXT (`.csv`, `.txt`).
4. Keep default selectors unless instructed otherwise.
5. Use one of these modes:
   - `Process Next Item` for one-by-one safe mode.
   - `Run Full Automation` for full run.
6. After review, click:
   - website button manually, or
   - extension `Click Final Confirm`.

## 4) Update to a new version

1. Download latest ZIP from GitHub.
2. Extract it.
3. Replace old local folder with new one.
4. Open `chrome://extensions`.
5. On Jobhandy card, click `Reload`.

## 5) Quick troubleshooting

- If extension does not run: make sure the company page tab is active.
- If buttons are disabled: upload file first.
- If Chrome shows extension load error: confirm selected folder contains `manifest.json`.
- If nothing happens after update: click `Reload` in `chrome://extensions` and refresh the company page.
